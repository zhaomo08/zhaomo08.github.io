什么是批处理程序
Batch file Programming 是微软操作系统自带的原生的开发语言,不需要构建任何环境就可以执行的脚本.

Batch file 批处理文件使用cmd.exe执行
如何编辑批处理程序
 使用任何文本编辑器
批处理程序可以做什么?
使用一系列内置命令进行自动化操作:
  1. 匹配规则删除文件
  2. 新建文件,日志
  3. 甚至可以批量创建计算机病毒
