编译

1.在项目目录下执行go build

2.在其他路径执行go build ,需要在后面加上项目的路径(项目路径从GOPATH/src后开始写起,编译之后的可执行文件就保存在当前目录下)

3.go build -o hello.exe //-o 设置build编译之后的名称

go run //像执行脚本文件一样执行go代码

go install //分为两步 先编译得到一个可执行文件 第二步 将可执行文件拷贝到 GOPATH/bin
